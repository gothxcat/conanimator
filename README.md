# conanimator

*Codename "Keith"*

Template for a terminal-based 2D text animator built on C++


## Build Dependencies

- GNU toolchain (>=C++11)


## Building

`./configure && make`


## Running

`./conanimator`


## Post-build Installation / Uninstallation

`sudo make install`

`sudo make uninstall`
