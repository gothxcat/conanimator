# conanimator

Template for a terminal-based 2D text animator


## Build Dependencies

- GNU toolchain (>=C++99)


## Building

`./configure && make`


## Running

`./conanimator`


## Post-build Installation / Uninstallation

`sudo make install`

`sudo make uninstall`
